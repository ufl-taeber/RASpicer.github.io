MetSign

Version: 1

Website

http://metaopen.sourceforge.net/metsign.html

Description

MetSign is designed for both the processing of LC-MS and DIMS data. It provides tools for pre-processing, annotation, puatative metabolite identification and clustering. There is the option of centralising profile data using either second-order polynomial fitting-based local maxima (SPF-LM) or one-dimensional discrete wavelet-transform (1-DWT). Centroid MS data are deconvoluted by detecting the XIC for each m/z (in favour of abundant peaks, with the those with the greatest peak matrix area being constructed first). To remove noise from the XIC, a wavelet transformation is used. A GGM is used to deconvolute overlapping peaks. An Exponenetially Modified Guassian (EMG) mixture model is used for peak fitting. For retention time alignment across samples, retention times are first transformed into z-scores and a reference peak list is used to align peaks, with peak distribution similarity being tested via a 2D-Kolmgorov Smirnov test. Data can then be normalised using quantile, cyclic loss or contrast-based normalisation. Metabolites are then putatively assigned using the MetSign database, which consists of KEGG, LIPID MAPS and HMDB. Fisher's exact test can be used to identify metabolites which are differentially expressed between treatment groups and FDR can then be used for correction. PCA and PLS can be used for dimensionality reduction for unsupervised clustering using either k-means, agglomerative or fuzzy C means clustering to visualise clustering between treatment groups.
